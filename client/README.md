# Frontend part for SPS project

## Project setup
```
npm install
```

## Copy environmental variables
```
cp .env.example .env
cp .env.production.example .env.production
```

## Compiles and hot-reloads for development
```
npm run serve
```

## Compiles and minifies for production
```
npm run build
```

## Lints and fixes files
```
npm run lint
```
